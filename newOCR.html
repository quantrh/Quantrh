How to setup & use this tool?

This web application acts as a front-end client for your local iOS OCR Server. It processes images directly on your local network using edge computing, ensuring 100% data privacy with zero cloud dependency.

  1. Download the core server app on your iPhone: OCR Server on App Store.
  2. Open the app on your iPhone and keep the screen on (Guided Access recommended).
  3. Note the IP address displayed on your iPhone screen.
  4. Enter that IP address below. Both devices must be on the same Wi-Fi network.
