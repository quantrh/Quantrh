OCR Server

Upload an image via upload API: 

curl -H "Accept: application/json" \
  -X POST http://<YOUR IP>:8000/upload \
  -F "file=@01.png"

OR

Upload an image via docOCR API: 

curl -H "Accept: application/json" \
  -X POST http://<YOUR IP>:8000/docOCR \
  -F "file=@01.png"
